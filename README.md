# File System

This project simulates a file system environment with permanent storage.
The system can be interacted from the terminal from a simple shell implementation.

## Dependencies

There are no dependencies

## Shell Commands

- exit
- logout
- mkdir
- touch
- ls
- cd
- clear
- rmdir
- echo
- cat
- help
